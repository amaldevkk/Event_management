# Event Management System

A web-based Event Management System built using Django that helps users manage events, registrations, and bookings with ease.

 # Features

- Create and manage events
- User registration & login system
- Event booking system
- Admin dashboard for event control
- Responsive UI design
- Secure authentication

# Technologies Used

- **Python**
- **Django**
- **HTML, CSS, JavaScript**
- **Bootstrap**
- **SQLite / PostgreSQL**
